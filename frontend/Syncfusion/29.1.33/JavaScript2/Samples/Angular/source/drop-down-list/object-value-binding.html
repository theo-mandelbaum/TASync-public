This sample demonstrates the object value binding functionalities of the DropDownList. Click the DropDownList element and select an item from the options list.The corresponding object value of the selected item is then assigned to the value property.

In the property panel, the value property of the selected item's will be displayed.

The DropDownList component contains a list of predefined values from that the user can choose a single value.

The object value binding sample illustrates the use of DropDownList that allows the end-users to select an item from the options list. Once an item is selected, its associated object value is automatically assigned to the value property. This behavior is enabled by the allowObjectBinding property.
