This sample demonstrates the itemTemplate functionalities of a ListBox.

The ListBox is a graphical user interface component used to display a list of items. This sample illustrates how to integrate the item template to customize the list item's content and this can be specified by using itemTemplate property.

In this sample, data is bound to the ListBox using the dataSource property.

More information about the ListBox can be found in the documentation section.
