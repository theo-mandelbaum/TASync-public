This sample demonstrates the basic rendering of the Angular Carousel component with items.

The Angular Carousel component is commonly used as a slideshow of images and contents. In this demo, simple slideshow of images has been showcased with the help of items property.

By default, the slide will be changed automatically with the interval of 5 seconds. You can also manually change the slide items using previous, next icons and indicators.

More information about Angular Carousel component can be found in this documentation section.
