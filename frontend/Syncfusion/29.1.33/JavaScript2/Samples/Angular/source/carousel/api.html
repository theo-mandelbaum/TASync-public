This sample demonstrates the properties available in the Angular Carousel component.

Carousel Properties
Enable Autoplay
Infinite Looping
Show Indicators

Show Arrows

In this demo, you can manually change the properties of the Angular Carousel component like autoPlay, buttonsVisibility, showIndicators, interval, showPlayButton, loop using the property panel.

More information about the properties available in the Carousel component can be found in this documentation section.
