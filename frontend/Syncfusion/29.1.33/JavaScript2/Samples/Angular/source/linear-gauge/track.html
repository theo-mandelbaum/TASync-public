            This sample demonstrates the basic axis, its edge style, range color for axis, inversed and opposed axis.

                roundedCornerRadius, 
                useRangeColor, 
                isInversed, and 
                opposedPosition properties can be used to set the edge style, range color for axis, inversed and opposed
            axis respectively.
