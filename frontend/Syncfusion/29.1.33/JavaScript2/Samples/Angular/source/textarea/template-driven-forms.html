This example demonstrates how to design template-driven form using TextArea component. The submit button will be enabled after filling the form with valid values. You can fill the details and submit the form.

The TextArea component is integrated with template-driven forms and it allows you to bind the value to model using ngModel. The textarea appearance will be changed based on its success or failure state. After the form is submitted, the values will be reset to defaults. The following validation is enabled in this example:

  • Email - required and valid email address
  • Comments - required
More information about template-driven forms is found in this reference link.
