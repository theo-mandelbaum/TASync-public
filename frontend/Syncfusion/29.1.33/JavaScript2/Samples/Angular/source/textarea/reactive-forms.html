Feedback Form

*This button is not a submit type and the form submit handled from externally.

This example demonstrates how to design Angular reactive form using TextArea component. The submit button will be enabled after filling the form with valid values. You can fill the details and submit the form.

The TextArea component is integrated with reactive forms by using formGroup. The textarea appearance will be changed based on its success or failure state. After the form is submitted, the values will be reset to defaults. The following validation is enabled in this sample:

  • Email - required and valid email address
  • Comments - required
More information about Angular reactive forms can be found in this reference link.
