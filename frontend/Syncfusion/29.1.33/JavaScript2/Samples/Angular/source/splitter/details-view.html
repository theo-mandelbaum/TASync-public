This example demonstrates the Splitter control that is used to design details view page. Select the employee from left pane to display the corresponding employee details on the right pane.

The Splitter is the layout user interface (UI) control, which integrates other JavaScript UI controls within its pane. In this example, the JavaScript ListView control is integrated within left pane to display the employee list and right pane to display the corresponding employee details.
