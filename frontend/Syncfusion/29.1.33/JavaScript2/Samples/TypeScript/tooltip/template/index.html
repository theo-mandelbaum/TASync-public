Description

This sample illustrates the way to display the template content on the tooltip. With the usage of Template, the user can format and structure the HTML content to be displayed on the tooltip as per their application needs. In this sample, the tooltip is integrated with toolbar component to display the respective icon’s information. Here, the HTML template design is compiled and then the resultant output display is directly assigned to the contentproperty of the tooltip. The template compilation process needs to be done on the beforeRender event of the tooltip. More information about setting template content on the Tooltip can be found in the documentation section.
