Description

The Dropdown Tree control loads the data through the dataSource property, where the data can be either local data or remote data. In case of local data, the data structure can be hierarchical data or list data (with self-referential format i.e., mapped with the value and parentValue fields). In this demo, the first Dropdown Tree is bound with the hierarchical data that contains the array of nested objects. And, the second Dropdown Tree is bound with the list type data where the parent-child relation is referred by the value and parentValue mapping fields.
