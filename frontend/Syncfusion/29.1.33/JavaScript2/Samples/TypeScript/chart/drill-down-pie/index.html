Description

In this example, you can see how to achieve the drilldown concept using a pie chart. Automobile sales are shown in different categories. By clicking each category, you can navigate to the next level, which shows the sales by categories made by each company. Datalabels are used in this sample to show information about the data points. Injecting Module Accumulation chart component features are segregated into individual feature-wise modules. To use datalabel, we need to inject DataLabel module using AccumulationChart.Inject(AccumulationDataLabel) method. More information on the pie series can be found in this documentation section.
