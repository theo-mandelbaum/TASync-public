This sample shows how to repeat a strip line in a chart.

In this example, you can see how to render and configure a strip line for the chart. To repeat the strip line, you need to set the startFromAxis, size, isRepeat, and repeatEvery properties accordingly in chartStripline.

Tooltip is enabled in this example, to see the tooltip in action, hover a point or tap on a point in touch enabled devices.

Injecting Module

Chart component features are segregated into individual feature-wise modules. To use column series, we need to inject Stripline module using Chart.Inject(Stripline) method.

More information on the strip line feature can be found in this documentation section.
