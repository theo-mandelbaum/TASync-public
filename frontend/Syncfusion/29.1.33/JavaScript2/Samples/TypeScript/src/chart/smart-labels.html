This sample shows the gold medal count scored by each country at the Rio Olympic Games using smart labels on the chart.

In this example, you can see how the labels can be arranged smartly without overlapping. You can use the EnableSmartLabels property to enable or disable the support.

Injecting Module

Accumulation chart component features are segregated into individual feature-wise modules. To use DataLabel, we need to inject AccumulationDataLabel using AccumulationChart.Inject(AccumulationDataLabel) method.

More information on the smart label in pie series can be found in this documentation section.
