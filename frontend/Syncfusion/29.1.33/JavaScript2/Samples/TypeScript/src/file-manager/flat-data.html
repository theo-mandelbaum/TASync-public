This sample demonstrates flat data rendering of the File Manager control. The File Manager uses a flat data object as an array of JSON objects for rendering, eliminating the need to define the ajaxSettings url.

The File Manager control is used to manage the files and folders in a file system. It supports all the basic file operations such as create, rename, delete, cut, copy, paste, upload, download, and more.

To load a folder data as an array of JSON objects, use the File Manager control fileSystemData property.
