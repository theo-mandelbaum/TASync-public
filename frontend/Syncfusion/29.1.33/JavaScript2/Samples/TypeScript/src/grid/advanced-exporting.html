This sample demonstrates the client-side exporting of the Grid, which allows you to export its data to the Excel, Pdf and CSV formats. Use the toolbar buttons to export Grid data with advanced options in same document to desired format.

Grid supports client-side exporting which allows you to export its data to the Excel, Pdf and CSV formats.

In this demo, excelexport and pdfexport items are defined in the toolbar. For these toolbar items, we have defined actions in toolbarClick event to export Grid with header and footer using the excelExport, pdfExport methods.


Injecting Module:

features are segregated into individual feature-wise modules. To use exporting feature, we need to inject ExcelExport and PdfExport module using the Grid.Inject(ExcelExport, PdfExport) method.
