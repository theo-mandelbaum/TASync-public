Description

The Dropdown Tree has been provided with several options to customize each list items, header, and footer elements. This sample uses the following list of templates in the Dropdown Tree ItemTemplate - To customize the list item's content. HeaderTemplate - To customize the header element. FooterTemplate - To customize the footer element. valueTemplate - To customize the selected item's element.
