Description

In this example, you can see how to print and export the rendered linear gauge. You can add print functionality by using the print method when allowPrint is set as true. Also, you can add export functionality by using the export method when allowImageExport and allowPdfExport are set as true. The linear gauge can be exported to JPEG, PNG, SVG, and PDF formats. More information about print and export can be found in this documentation section.
