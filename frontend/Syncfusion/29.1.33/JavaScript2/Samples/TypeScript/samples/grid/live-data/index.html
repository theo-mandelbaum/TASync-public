Description

The updating of Grid cells can be done without any performance lagging by using the setCellValue method. The style of Grid cells can be customized using the queryCellInfo event. In this demo, Clicking the start update button triggers automatic updates of Grid cells at the interval set in the feed delay text box which is a milliseconds format. Clicking the stop update button will halt the automatic updating of Grid cells. More information on the queryCellInfo can be found in this documentation section.
