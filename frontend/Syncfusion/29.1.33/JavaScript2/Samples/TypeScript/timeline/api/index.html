Description

This sample can be customized further with the combination of Timeline properties from the property pane. For example, The layout can be changed by selecting the orientation dropdownlist from property pane. Items alignment can be changed by selecting the alignment dropdownlist from property pane. This can be achieved by using the align property. Show or hide the information opposite to the content by toggling the opposite content switcher button. Show or hide the item icons by toggling the show icon switcher button. Reverse the timeline items by toggling the reverse switcher button. This can be achieved by using the reverse property.
