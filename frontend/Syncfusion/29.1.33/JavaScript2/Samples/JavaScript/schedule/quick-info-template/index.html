Description

In this demo, the quick popup is customized based on the office required appointment-related fields which can be achieved by making use of the quickInfoTemplate option. The quickInfoTemplate has three UI elements such as header, content, and footer. You can customize these UI elements of the quick popup. You can also customize whether the quick popup is applicable to the cells or events or for both using the elementType property.
