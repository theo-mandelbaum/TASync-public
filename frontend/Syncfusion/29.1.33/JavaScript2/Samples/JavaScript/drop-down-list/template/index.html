More Details...

Description

The DropDownList has been provided with several options to customize each list items, group title, selected value, header and footer elements. This sample uses the following list of templates in DropDownList ItemTemplate - To customize the list item's content. ValueTemplate - To customize the value element content that holds the selected item's text. HeaderTemplate - To customize the header element. More information on the template feature configuration can be found in the documentation section.
