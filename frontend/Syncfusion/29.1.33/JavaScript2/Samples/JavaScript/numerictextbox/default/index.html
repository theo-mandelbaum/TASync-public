Description

The NumericTextBox component is used to get the number inputs from the user. The input values can be increased or decreased with a predefined step value. In this demo, the default NumericTextBox is rendered with the percent and currency formats as specified in MSDN. More information on the NumericTextBox instantiation can be found in the documentation section.
