Description

This example showcases the implementation of redaction using rectangle annotation, leveraging the Redaction feature in File formats. Various types of redactions are demonstrated: Displaying text over the redacted area Adding an image to the redacted area Drawing patterns on the redacted area Blacking out the redacted area Whitening out the redacted area Upon selecting areas on a page or across different pages with different redaction options, users can click the `Redact button`. This action redacts the document and reloads it into the PDF Viewer. The redacted document can then be saved or downloaded. More information on adding annotation programmatically can be found in this documentation section.
