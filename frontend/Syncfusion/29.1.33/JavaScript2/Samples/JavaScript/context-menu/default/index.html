This sample demonstrates the default functionalities of the ContextMenu. Right click/Touch hold the rectangular area to open the ContextMenu.

More Details...

    Description

    ContextMenu is a graphical user interface that appears on the user right click/touch hold action. It has support to provide single level/multiple level of ContextMenu. In this demo, target property is set as '#contextmenutarget'. Hence, on right clicking the target element, the ContextMenu will open. More information about ContextMenu can be found in this documentation section.
