This sample demonstrates the real-time use case of File Manager in a web application. Dialog and Uploader controls are integrated with the File Manager. Click the browse button in the uploader element to open the File Manager inside the Dialog control. 

The File Manager control is used to explore a file system through a web application, similar to the Windows Explorer for Windows. It supports all the basic file operations such as create, rename, delete, cut, copy, paste, upload, download, and more.

Note: File Manager’s upload functionality is restricted in the online demos for security reasons. If you need to test upload functionality, please install Syncfusion Essential Studio on your machine and run the demo.
