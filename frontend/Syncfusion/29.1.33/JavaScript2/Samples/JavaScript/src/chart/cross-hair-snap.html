This sample demonstrates the crosshair with snap-to-data functionality in charts. Hover over the chart or tap on it in touch-enabled devices to view the crosshair and its tooltip snapping directly to the nearest data point.

The crosshair in charts helps users examine data values with precision by using a vertical and horizontal line. When the snapToData property is enabled, the crosshair aligns directly to the nearest data point, making it easy to pinpoint exact values on hover or tap. Enable this feature by setting snapToData: true in the crosshairSettings configuration.

Injecting Module

Chart component features are segregated into individual feature-wise modules. To use Crosshair, we need to inject Crosshair module using Chart.Inject(Crosshair) method.
