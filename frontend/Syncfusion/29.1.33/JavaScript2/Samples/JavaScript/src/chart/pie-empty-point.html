Empty Point Mode:

This sample illustrates the annual product-wise profit analysis of an organization with empty point functionality in the pie series. The Mode of empty point can be changed by using Empty Point Mode in property panel.

In this example, you can see how to render and configure the empty points. The empty point in the chart can be handled using the EmptyPointSettings property. Tooltip is enabled in this example, to see the tooltip in action, hover a point or tap on a point in touch enabled devices.


More information on the empty point in pie series can be found in this documentation section.
