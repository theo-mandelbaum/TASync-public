This sample demonstrates the integration of the Dropdown List and Slider controls as templates in the Query Builder control and also showing the different types of queries such as CEL and SpEL. The query preview can be changed using the tab control.

This sample illustrates how to integrate the Dropdown List, Slider, and Checkbox controls with the Query Builder. The available types of templates are:

  • DropDownList
  • AutoComplete
  • CheckBox
  • Slider

In this demo queries are exported and imported in CEL and SpEL formats. For Common Expression Language (CEL) output, use the "cel" format. CEL is used for validating data. For Spring Expression Language (SpEL) output, use the "spel" format. The Spring Expression Language (SpEL) is a powerful expression language that supports querying and manipulating an object graph at runtime.
