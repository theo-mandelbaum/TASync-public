This sample demonstrates the template functionality of the JavaScript Message control. Click the dismiss button to hide the message. Click the Show pull request button to restore the hidden message.

The Message control has an option to customize the content with a custom structure. The content can be a string, paragraph, or any other HTML element.

In this sample, the Message control content is customized with HTML elements and JavaScript Button controls.

More information about Message template can be found in this documentation section.
