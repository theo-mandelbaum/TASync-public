This sample demonstrates, the functionality of dynamically adding the panels inside the DashboardLayout component. Go to the properties panel section at the right side and select any pre-designed layout, based on selection the panles are rendered inside the DashboardLayout. Click the `Reset` button to remove the panels from the component.

By default this sample rendered as empty area. On selecting any pre-designed layout available in the properties panel will be appended inside the DashboardLayout.
