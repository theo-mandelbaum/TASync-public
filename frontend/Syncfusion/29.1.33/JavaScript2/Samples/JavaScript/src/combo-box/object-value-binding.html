This sample demonstrates the object value binding functionalities of the ComboBox. Type a character in the ComboBox element or click the drodown icon to choose an item from the options list. The corresponding object value of the selected item is then assigned to the value property. In the property panel, the value property of the selected item's will be displayed.

The ComboBox component allows the user to type a value, or choose an option from the list of predefined options.Once an item is selected, its associated object value is automatically assigned to the value property. This behavior is enabled by the allowObjectBinding property.
