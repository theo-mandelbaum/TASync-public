National Sports

Description

This sample illustrates the way to load the content of a tooltip dynamically using AJAX request. Here, when the user hovers/tap on the country names, its respective data (national game of each country and its related game icon) will be retrieved dynamically and then assigned to the tooltipâ€™s content. The AJAX request should be made within the beforeRender event of the tooltip, and on every success, the corresponding retrieved data will be set to the content property of the tooltip. More information on loading dynamic tooltip content can be found in the documentation section.
