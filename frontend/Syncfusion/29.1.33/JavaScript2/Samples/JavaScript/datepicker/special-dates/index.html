Description

Special Dates sample demonstrates, how to customize a specific dates in a datepicker by using renderDayCell event. This event gets triggered on each day cell element creation that allows you to customize or disable the specific dates in datepicker. Here 10, 15 and 25 date's are customized with custom styles and custom classes. More information on the DatePicker customization can be found in the documentation section.
